Basic OpenCart AutoComplete for PHPStorm
-
**Based on OpenCart version 3.0.3.2**


Install Steps
-
Open **File > Settings... (Ctrl+Alt+S) > Language & Frameworks > PHP** and add the directory "**OpenCart Autocomplete**" in yours **Include Path**
